The Web & How It Works

The web is the world's largetst network of computers. Essentially, it can be best described as a bunch of computers that communicate with each other.

It is mostly make up of a collection of HTML documents; however, any type of file can be linked to the web. Some other examples of files that can be found on the web are: Plain Text, HTML, Images, Videos, and Music.

HTML or Hyper Text Markup Language documents are the main type of content on the web. HTML documents are made up of:

- Text Content (what you see)
- Markup (what it looks like)
- Links to other web pages
- References to other documents

HTTP is the main protocol of the web. When a person goes to a web page like www.google.com, their computer sends a HTTP request to a server, or computer that hosts the files that makeup the web. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser such as FireFox, Internet Explorer, or Chrome interprets the page and displays it on the user's screen. (A webpage is a text document written in HTML.)

HTML Jargon

Tag

- Contained in angled brackets
- Most have an opening and closing tag

Void Tag

- Do not require a closing tag because they have no content - Images and page breaks are examples of Void Tags

Element
- Refers to everything written within a set of opening and closing tags
- Inline vs. Block Elements: Block elements form an "invisible box" around the content inside them.

Attribute
- Is a property of a HTML element

Things to Remember

Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote. Computers are stupid. Computers interpret instructions literally which makes them very unforgiving. A small mistake by a programmer can cause huge problems in a program so programmers must write code exactly exactly the way a computer understands. (Writing with correct syntax.)

Programmers cannot remember everything. There are simply too many details; however, it is very easy to search for and find what you are looking for.
